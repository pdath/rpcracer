# Vendored Dependencies

This directory contains vendored third-party libraries:

- `yyjson.h` / `yyjson.c` - JSON parser (v0.12.0, MIT license)
- `uthash.h` - Hash table macros (v2.3.0, BSD license)

These files are committed to the repository. No download step is needed to build.

To update to newer versions, edit and run `./vendor.sh` from the project root.
